To execute the program: `make run`

Input Bindings:

Translation - Left mouse button + drag
Rotation - Right mouse button + drag
Selecting object - Ctrl + Left mouse button / Number keys
Unselecting object - Escape
Scaling - Plus / Minus keys
Toggle wireframe - w key
Toggle normal coloring - n key
Toggle splats - s key
Increase splat radius - k key
Decrease splat radius - j key
